  <how-to-verify>
**Test 1 -- iOS Safari file picker (UPLD-01):**
1. Open the site on an iPhone (use the preview URL or local network URL)
2. Scroll to "Share Your Story" section
3. Tap the upload drop zone
4. Verify: iOS presents options for Camera, Photo Library, and Browse (Files app)
5. Select a video from Camera Roll
6. Verify: file appears in the file list with name and size

**Test 2 -- Desktop drag-and-drop (UPLD-02):**
1. Open the site in Chrome/Firefox on desktop
2. Drag a video file from Finder/Explorer onto the drop zone
3. Verify: drop zone border turns red (#B91C1C) during drag hover
4. Drop the file
5. Verify: file appears in the file list

**Test 3 -- Progress bar (UPLD-03):**
1. Select a large video file (50+ MB for visible progress)
2. Click "SUBMIT YOUR STORY"
3. Verify: button text changes to "UPLOADING..."
4. Verify: progress bar appears and fills from left to right
5. Verify: percentage label updates in real time
6. Verify: on completion, progress reaches 100%

**Test 4 -- Error handling (UPLD-04):**
1. Turn off WiFi/network on the device
2. Select a file and click submit
3. Verify: error banner appears with a message and "Try again" link
4. Turn WiFi back on
5. Click "Try again"
6. Verify: upload retries and succeeds

**Test 5 -- File type validation (UPLD-05):**
1. Try to select a PDF or image file via the file picker
2. Verify: error message appears: "That file type isn't supported. Please select a video or audio file."
3. The upload does NOT start
